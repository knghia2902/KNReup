# Phase 3, Plan 1: UI Foundation & NLE Layout Shell

<objective>
Lột xác toàn bộ hệ thống CSS cũ và thiết lập Layout Shell mới (NLELayout) dựa trên bản thiết kế tĩnh `knreup-complete.html`. Chia tách các vùng hoạt động Titlebar, Sidebar, Module Workspaces, Timeline tĩnh thành React components riêng biệt.
</objective>

<requirements>
- Thiết lập CSS Variables và Global Styles mới (`--w0`, `--i0`, .etc)
- App layout skeleton: Titlebar + Sidebar + Media Bin + Preview + Properties + Timeline
- Sidebar state switching (UI tĩnh trước)
</requirements>
