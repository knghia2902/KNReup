## Summary

Triển khai thiết kế Stitch UI cao cấp và tính năng Đăng nhập Douyin qua WebView.

### What was built

1. **Stitch CSS Refactor** (`downloader.css`) — Chuyển đổi toàn bộ hệ thống variable sang prefix `--dl-`, áp dụng font `Outfit`, layout grid bất đối xứng (1.1fr 0.9fr), và bo góc card `2.5rem`.
2. **DouyinAuthWidget** (`DouyinAuthWidget.tsx`) — Component mới thay thế `CookieManager`, hỗ trợ mở cửa sổ WebView đăng nhập Douyin trực tiếp trong app Tauri qua `@tauri-apps/api/webviewWindow`.
3. **Tactile React Components** — Cập nhật `URLInput` và `DownloadOptions` với hiệu ứng tactile phản hồi tốt hơn, màu sắc lỗi Deep Rose (`#E11D48`), và các cascading animations cho danh sách format.
