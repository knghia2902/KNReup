## Accomplishments
- Implemented visual highlight and standard active state mapping across video, audio, bgm, and subtitle tracks.
- Implemented `Ctrl + B` for splitting subtitle clips exactly at the playhead position.
- Expanded dragging functionality for all background media strips, featuring magnetic playhead/snap support.
- Implemented `Delete` and `Backspace` to ripple-delete subtitle clips, shifting trailing segments automatically.

## Files Created/Modified
- `src/stores/useProjectStore.ts` - Added global `selectedClipId` tracking.
- `src/stores/useSubtitleStore.ts` - Added `deleteAndRippleSubtitle` reducer for advanced array tracking.

## Decisions Made
- Centralized global keyboard events (`Ctrl + B`, `Delete`) in `Timeline.tsx` instead of having individual tracks handle DOM focus, allowing consistent operation from hotkeys.
- Selected clip uses `sub-${id}`, `vid-main`, `aud-main`, `bgm-main` format in `useProjectStore` to maintain unified context while preserving local `SubtitleStore` id selection.
