**Cấu trúc component:**
```
VoiceCloneWindow (root)
├── HeaderBar (draggable, icon + title + connection status)
├── TabBar (3 tabs: Clone Giọng, Thiết Kế Giọng, Profiles)
├── TabContent
│   ├── CloneTab
│   │   ├── DropZone (drag & drop + file dialog)
│   │   ├── AudioInfo (filename, duration, format)
│   │   ├── WarningBanner (>10s) / ErrorBanner (>30s)
│   │   ├── ComparisonPlayer (gốc vs clone — D-08)
│   │   ├── NameInput
│   │   └── CTA "Clone Voice"
│   ├── DesignTab
│   │   ├── DescriptionTextarea
│   │   ├── QuickAttributeChips (Region, Giọng, Tuổi - UI dạng Chip/Pill)
│   │   ├── SampleTextDropdown + CustomTextInput
│   │   ├── PreviewPlayer (với Progress Bar)
│   │   ├── NameInput
│   │   └── CTA "Design Voice"
│   ├── ProfilesTab
│   │   ├── ProfileList (vertical, mỗi profile 1 dòng — D-04, D-05)
│   │   │   └── ProfileItem (tên + ngày + duration + Play + Delete)
│   │   └── EmptyState
│   └── HistoryTab (hoặc Lịch sử TTS)
│       └── HistoryList (list preview audio)
└── ConfirmDialog (xóa profile)
```

**Yêu cầu kỹ thuật:**
- Dùng `useSidecar` hook để giao tiếp sidecar API
- State quản lý: activeTab, selectedFile, fileInfo, cloneResult, profiles, history, isCloning, isDesigning
- Upload qua `fetch` POST multipart/form-data đến `/tts/profiles/clone`
- Design qua `fetch` POST JSON đến `/tts/profiles/design` (cần gửi thêm tham số `region`, `speed` từ state)
- Preview qua `fetch` POST JSON đến `/tts/profiles/preview`
- List profiles qua `fetch` GET `/tts/profiles/`
- Delete qua `fetch` DELETE `/tts/profiles/{name}`
- Audio playback dùng HTML5 `<audio>` element (không cần wavesurfer)
- File validation: chỉ chấp nhận `.wav`, `.mp3`, `.ogg`, `.flac`, `.m4a`
- Duration validation client-side: >10s → warning banner, >30s → error + disable CTA
- Câu mẫu tiếng Việt sẵn cho preview (D-07):
  - "Xin chào, tôi là trợ lý AI của KNReup"
  - "Hôm nay trời đẹp quá, chúng ta đi dạo nhé"
  - "Video này được tạo bởi công nghệ AI hiện đại"
- Export component name: `VoiceCloneWindow`
- Import và dùng Phosphor Icons: Microphone, Play, Pause, Trash, Upload, MagicWand
