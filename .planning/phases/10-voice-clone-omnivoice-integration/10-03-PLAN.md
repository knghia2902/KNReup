# Plan 03: TTS Pipeline Integration — Giọng Clone trong Editor Dropdown

<objective>
Tích hợp giọng clone vào TTS dropdown của Editor (D-09). Khi user tạo giọng ở Voice Clone tool, giọng đó xuất hiện trong danh sách voice của TTS tab, cùng với Edge TTS, gTTS, ElevenLabs. User chọn giọng clone → pipeline dùng OmniVoice engine để synthesize.
</objective>

<tasks>

## Task 1: Cập nhật TTSTab — fetch và hiển thị giọng clone

<read_first>
- src/components/properties/TTSTab.tsx
- python-sidecar/app/routes/tts_profiles.py
- python-sidecar/app/engines/tts/omnivoice_engine.py
</read_first>

<action>
Cập nhật TTSTab.tsx để:

1. **Fetch danh sách giọng clone từ API** khi component mount:
   ```typescript
   // Thêm state cho cloned voices
   const [clonedVoices, setClonedVoices] = useState<Array<{name: string, id: string, type: string}>>([]);
   
   useEffect(() => {
     fetch(`${sidecarUrl}/tts/profiles/`)
       .then(r => r.json())
       .then(data => setClonedVoices(data.profiles || []))
       .catch(() => setClonedVoices([]));
   }, [sidecarUrl]);
   ```

2. **Merge vào voice dropdown** — Thêm optgroup "Cloned Voices" vào dropdown list hiện có:
   ```tsx
   {/* Trong voice select/dropdown */}
   <optgroup label="Cloned Voices">
     {clonedVoices.map(v => (
       <option key={v.id} value={`omnivoice:${v.id}`}>
         🎤 {v.name}
       </option>
     ))}
   </optgroup>
   ```

3. **Xử lý voice selection** — Khi user chọn giọng clone:
   - Voice value format: `omnivoice:{profile_name}`
   - Parse prefix để xác định engine = "omnivoice"
   - Truyền profile_name vào synthesize request

4. **Refresh mechanism** — Thêm nút refresh hoặc auto-refresh khi window regains focus:
   ```typescript
   useEffect(() => {
     const handleFocus = () => {
       fetch(`${sidecarUrl}/tts/profiles/`)
         .then(r => r.json())
         .then(data => setClonedVoices(data.profiles || []))
         .catch(() => {});
     };
     window.addEventListener('focus', handleFocus);
     return () => window.removeEventListener('focus', handleFocus);
   }, [sidecarUrl]);
   ```

5. **Visual indicator** — Giọng clone hiển thị icon 🎤 prefix để phân biệt với system voices.
</action>

<acceptance_criteria>
