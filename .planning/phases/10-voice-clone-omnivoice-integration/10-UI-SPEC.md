## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding, tag pills |
| sm | 8px | Compact element spacing, button gaps |
| md | 16px | Default element spacing, section padding |
| lg | 20px | Window content padding (matches Downloader) |
| xl | 32px | Major section breaks |
| 2xl | 48px | Empty state icon-to-text gap |

Exceptions:
- Header padding: `16px 20px` (vertical/horizontal) — follows DownloaderWindow pattern, adjusted from 14px to maintain 4px grid
- Input container padding: `8px 12px` — matches DownloaderWindow input field (12px = 3×4, valid multiple)

## Typography

| Role | Size | Weight | Line Height | Font |
|------|------|--------|-------------|------|
| Caption / Label | 10px | 400 (regular) | 1.3 | Geist Mono |
| Body / Button / Tab | 13px | 400 (regular) | 1.5 | Geist |
| Heading | 15px | 600 (semibold) | 1.3 | Geist |

Weights: **400** (regular — body, labels, captions) and **600** (semibold — headings, CTAs, active tabs).

> Source: design-system.css `--font-sans: 'Geist'`, `--font-mono: 'Geist Mono'`. 4-size scale: 10 (caption) → 13 (body) → 15 (heading) → 20 (display). Only 2 weights: 400 and 600.

### Interaction Patterns

| Interaction | Behavior |
|-------------|----------|
| File upload | Drag & drop onto drop zone OR click to open file dialog |
| Audio >10s warning | Yellow banner with `var(--warning)` color, non-blocking |
| Audio >30s reject | Error banner, disable clone button |
| Clone progress | Linear progress bar with percentage, disable CTA |
| Profile play | Inline audio player in profile row, one at a time |
| Profile delete | Confirmation dialog: "Xóa {name}?" with Cancel/Delete |
| Tab switch | Instant, no animation. Active tab has `var(--accent)` underline |
| Preview comparison | Side-by-side audio players, original left / clone right (D-08) |

## State Mapping

| UI State | Visual Treatment |
|----------|-----------------|
| Idle / Empty | Empty state illustration + copy |
| File Loaded | Audio info card + preview section visible |
| Cloning / Generating | CTA disabled, ProgressOverlay visible with percent |
| Clone Success | Preview comparison section populated, "Lưu Profile" option |
| Clone Error | Error banner with retry suggestion |
| Engine Offline | Full-window overlay: "OmniVoice chưa sẵn sàng" + Settings link |
| Warning (>10s) | Yellow warning banner, clone button still enabled |
| Rejected (>30s) | Red error banner, clone button disabled |
