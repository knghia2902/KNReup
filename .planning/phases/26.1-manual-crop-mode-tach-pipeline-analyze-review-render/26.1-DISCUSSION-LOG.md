## Tracking Data Format & Storage

| Option | Description | Selected |
|--------|-------------|----------|
| JSON cạnh video gốc | `A.mp4` → `A_tracking.json`, dễ tìm, di chuyển cùng video | ✓ |
| Thư mục riêng | `data/tracking/{hash}/`, gọn nhưng khó tìm | |

**User's choice:** JSON cạnh video gốc
**Notes:** Tracking data ~50-200KB/video, nhẹ. localStorage chỉ lưu metadata pointer.
