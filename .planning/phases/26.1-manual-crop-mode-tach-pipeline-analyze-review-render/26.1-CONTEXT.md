# Phase 26.1: Manual Crop Mode — Tách pipeline Analyze → Review → Render - Context

**Gathered:** 2026-04-30
**Status:** Ready for planning

<domain>
## Phase Boundary

Thêm chế độ Manual Crop cho Smart Crop tool hiện có. Tách pipeline `process()` monolithic thành 3 bước:
1. **Analyze** — Chạy AI YOLOv8 detection + tracking, xuất tracking data JSON (không render video)
2. **Review** — User xem crop overlay trên video gốc, kéo/chỉnh vị trí crop, tạo keyframes
3. **Render** — Đọc tracking data (+ keyframe edits) → crop video → xuất output

Cho phép re-render mà không cần chạy lại AI. Giữ nguyên mode Auto hiện tại — Manual là option bổ sung.

**Không bao gồm:** Scene split detection, timeline minimap, batch manual crop, thay đổi pipeline Render trong Editor.

</domain>

<decisions>
## Implementation Decisions

### Tracking Data Format & Storage
- **D-01:** Tracking data lưu dạng JSON cạnh video gốc: `A.mp4` → `A_tracking.json`
- **D-02:** File JSON chứa: version, source path, fps, resolution, crop_size, settings, per-frame data (idx, cx, cy, faces, speaker_id), và mảng keyframes (user edits)
- **D-03:** localStorage (Zustand persist) chỉ lưu metadata pointer (path tới tracking JSON), không lưu data nặng
- **D-04:** Tracking data ~50-200KB cho video 5 phút — nhẹ, di chuyển cùng thư mục video

### Review UI — Crop Overlay
- **D-05:** Hiển thị crop rectangle overlay (Canvas) trên video gốc bên panel Before — user thấy vùng đang crop
- **D-06:** User kéo/drag rectangle theo chiều ngang = chỉnh crop position tại frame hiện tại → tạo keyframe
- **D-07:** Click frame → set keyframe, delete keyframe → quay về vị trí AI tracking
- **D-08:** Chỉ hỗ trợ horizontal pan (kéo ngang) — KHÔNG zoom/rotation
- **D-09:** Timeline minimap (đường đi crop path) là deferred — không làm trong phase này

### Workflow Flow — UI Layout
- **D-10:** Giữ layout hiện tại, thêm mode toggle **"Auto / Manual"** ở header SmartCropWindow
- **D-11:** Mode Auto (mặc định) = flow hiện tại: chọn file → Export → done (không thay đổi gì)
- **D-12:** Mode Manual = chọn file → nút [Analyze] → review overlay → nút [Render]
- **D-13:** Action button thay đổi theo stage: `[▶ Analyze]` → `[đang review...]` → `[⬇ Render]`
- **D-14:** Nút "Re-analyze" cho phép chạy lại AI khi cần (đổi settings)

### Re-render & Settings Override
- **D-15:** Cho phép đổi output resolution (1080×1920 / 720×1280) trước khi render
- **D-16:** Cho phép đổi CRF (chất lượng encode) trước khi render
- **D-17:** Luôn render toàn bộ video — KHÔNG partial render (preview overlay đủ để check)

### Agent's Discretion
- Chi tiết kỹ thuật Canvas overlay implementation (resize, coordinate mapping)
- Keyframe interpolation algorithm (linear interpolation giữa keyframes)
- Tracking JSON file format chi tiết (số decimal, compression)
- UX transitions giữa Analyze → Review → Render stages

</decisions>
