<decisions>
## Implementation Decisions

### Kiến trúc Main Track
- **D-01:** Main Track kiểu CapCut — Main Track nằm ở giữa Timeline, là track lớn nhất và nổi bật nhất. Cho phép xếp nhiều video clip nối tiếp nhau trên cùng 1 track.
- **D-02:** Track layout từ trên xuống dưới:
  ```
  [Overlay zone — text/sticker tracks tự do (tương lai)]
     SUB track (phụ đề — cố định, nằm trên Main)
  ─── 🎬 MAIN TRACK (multi video clip nối tiếp) ───
     TTS track (giọng đọc AI — cố định, nằm dưới Main)
     BGM track (nhạc nền — cố định, nằm dưới Main)
  [Thêm audio tracks tự do bên dưới (tương lai)]
  ```
- **D-03:** SUB track nằm phía trên Main Track vì phụ đề là "overlay" lên video. TTS/BGM nằm dưới Main vì là audio.

### Data Model — Clip Model Thống Nhất
- **D-04:** Chuyển **hoàn toàn** sang clip model cho tất cả loại content (video, audio, subtitle). Mọi thứ trên timeline đều là Clip.
- **D-05:** Clip data structure:
  ```typescript
  Clip {
    id: string;
    trackId: string;       // "main" | "sub" | "tts" | "bgm"
    type: "video" | "audio" | "text" | "subtitle";
    sourceFile: string;    // đường dẫn file gốc
    sourceStart: number;   // điểm bắt đầu cắt trong file gốc
    sourceDuration: number;
    timelineStart: number; // vị trí trên timeline
    timelineDuration: number;
  }
  ```
- **D-06:** Subtitle clips mở rộng thêm: `source_text`, `translated_text`, `confidence`, `tts_status`, `tts_audio_path` — giữ liên kết sub↔TTS qua `tts_audio_path`.
- **D-07:** Pipeline Whisper/Translate vẫn xuất segments JSON → khi đưa vào Timeline thì chuyển đổi thành SubtitleClip.

### Hành vi Track
- **D-08:** Ripple Delete mặc định — xóa clip trên mọi track → các clip phía sau tự dồn lên lấp khoảng trống.
- **D-09:** Mỗi track có control panel bên trái giống CapCut: lock 🔒, visibility 👁️, volume 🔊, menu ⋯.

### Drag-Drop (Mức 1 — cơ bản)
- **D-10:** Kéo video từ Media Bin → đặt nối đuôi clip cuối cùng trên Main Track.
- **D-11:** Kéo audio từ Media Bin → đặt trên BGM track.

### Agent's Discretion
- Chi tiết kiến trúc `useTimelineStore` (track registry, clip registry)
- Cách chia tách component Timeline.tsx hiện tại (652 dòng monolithic)
- Cách migrate data cũ (`useProjectStore` vid_clip_start/vid_clip_duration) sang clip model mới
- Auto-scroll khi kéo file sát mép timeline

</decisions>
