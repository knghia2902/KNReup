**Ngày**: 2026-04-27
**Phase**: 08.3

## Test Cases

### T-01: Timeline Layout 4 Tracks
**Mô tả**: Mở Editor → load video → kiểm tra timeline hiển thị 4 track rows
**Bước**:
1. Từ trang chủ, chọn dự án có video
2. Vào Editor → cuộn xuống khu vực Timeline
3. Verify layout: SUB (trên) → MAIN (60px, nổi bật) → TTS → BGM (dưới)

**Expected**: 4 track rows hiển thị đúng thứ tự, Main track lớn hơn các track khác
**Status**: ⏳ Chờ user
