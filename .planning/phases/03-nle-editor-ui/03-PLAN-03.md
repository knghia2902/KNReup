**Structure:**
```
<Section title="Segments">
  <!-- Segment list (scrollable) -->
  <div className="segment-list">
    {segments.map(seg => (
      <div className={`segment-item ${selected ? 'active' : ''}`}>
        <span className="seg-time mono">{formatTime(seg.start)}</span>
        <span className="seg-text">{seg.translated_text || seg.source_text}</span>
        <ConfidenceBadge value={seg.confidence} />
      </div>
    ))}
  </div>

  <!-- Selected segment editor -->
  {selectedSegment && (
    <div className="segment-editor">
      <Row>
        <TimecodeInput label="Start" value={seg.start} onChange={...} />
        <TimecodeInput label="End" value={seg.end} onChange={...} />
      </Row>
      <div className="seg-source">{seg.source_text}</div>
      <textarea className="seg-translated" value={seg.translated_text} onChange={...} />
      <div className="seg-actions">
        <button>Split</button>
        <button>Merge</button>
        <button>Delete</button>
        <button>Re-TTS</button>
      </div>
    </div>
  )}
</Section>
```

**ConfidenceBadge:**
- `confidence > 0.8` → `badge-success` (green)
- `confidence 0.5-0.8` → `badge-warning` (yellow)
- `confidence < 0.5` → `badge-danger` (red)
- Hiển thị: `Math.round(confidence * 100)%`

**TimecodeInput:**
- Format: `MM:SS.ms` (e.g., "01:23.45")
- Monospace font (cockpit mode)
- Input type text, parse on blur
- Width: 64px

**Segment Item styling (taste-skill):**
- `min-height: 28px`, `padding: 3px 8px`
- Border-bottom: `1px solid rgba(255,255,255,0.03)`
- Active: `background: var(--accent-subtle)`, left border `2px solid var(--accent)`
- Time: `font-family: var(--font-mono)`, `font-size: 10px`, `color: var(--text-muted)`
- Text: `font-size: 11px`, truncate with ellipsis

**Action buttons:**
- styled: `btn` class, small size, Phosphor icons size 12
- Split: Scissors icon, Merge: ArrowsInSimple icon, Delete: Trash icon, Re-TTS: Microphone icon
- Re-TTS: calls `POST /api/pipeline/tts-segment` with segment data

**Search/Replace:**
- 2 text inputs: "Tìm" + "Thay thế"
- Button "Thay tất cả" — replaces in all segments' translated_text
- Compact: font-size 11px, var(--bg-surface) background

**SRT Import/Export buttons** (bottom):
- "Import SRT": opens file dialog, parses SRT via subsrt-ts, calls `setSegments()`
- "Export SRT": builds SRT string via subsrt-ts, saves via Tauri save dialog
- Styled: `btn` class

All actions connected to `useSubtitleStore`.
Empty state khi chưa có segments: "Chưa có subtitle — upload video hoặc import SRT"
