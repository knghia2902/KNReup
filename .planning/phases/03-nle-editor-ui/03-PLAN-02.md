Tạo `src/components/editor/VideoPreview.tsx`:

**Structure:**
```
<div className="video-preview-container">
  <div className="video-viewport" ref={viewportRef}>
    <!-- Empty state khi chưa có video -->
    <div className="preview-empty">
      <Play size={48} weight="thin" />
      <span>No media loaded</span>
    </div>

    <!-- Video + Canvas khi có video -->
    <video ref={videoRef} style={videoStyle} />
    <canvas ref={canvasRef} style={canvasStyle} />
  </div>
  <VideoControls videoRef={videoRef} />
</div>
```

**Props:**
```typescript
interface VideoPreviewProps {
  videoSrc: string | null;       // Tauri convertFileSrc URL or null
  segments: SubtitleSegment[];   // từ useSubtitleStore
  subtitleConfig: {              // từ useProjectStore
    enabled: boolean;
    position: number;
    font_size: number;
    font: string;
    color: string;
    outline_color: string;
  };
}
```

**Canvas Rendering Loop:**
1. `useEffect` với `requestAnimationFrame` loop khi video đang play
2. Mỗi frame: clear canvas → check `video.currentTime` → find matching segment → draw text
3. Text rendering: `ctx.font`, `ctx.fillStyle`, `ctx.strokeStyle`, `ctx.fillText()`, `ctx.strokeText()`
4. Position mapping: `subtitle_position` 1-5 → Y coordinate (10%, 30%, 50%, 70%, 90% of canvas height)
5. Text centering: `ctx.measureText(text).width` → center X
6. Vietnamese line height: extra 20% padding cho diacritics

**Responsive:**
1. `ResizeObserver` trên viewport container
2. Calculate letterbox dimensions based on video native ratio
3. Canvas width/height = letterbox dimensions
4. CSS: `object-fit: contain` trên video element

**Font Loading:**
1. Load font via `FontFaceObserver` khi `subtitleConfig.font` thay đổi
2. Test string: 'Ẩặẫậ' (Vietnamese diacritics)
3. On success: set font ready flag, re-render canvas
4. On fail: fallback to system sans-serif

**Empty State:**
- Khi `videoSrc` null: hiển thị Play icon + "No media loaded" text
- Style: centered, opacity 0.25, Phosphor Play icon weight="thin" size=48
- Pattern giống preview-empty hiện có trong NLELayout.css
