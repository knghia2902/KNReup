<decisions>
## Implementation Decisions

### Dynamic Overlay Tracks
- **D-01:** Hỗ trợ dynamic overlay tracks — không giới hạn số lượng. Mỗi lần user kéo video lên phía trên Main Track → tự tạo overlay track mới (`overlay-1`, `overlay-2`...).
- **D-02:** `TrackId` type cần mở rộng từ union cố định sang hỗ trợ string pattern `overlay-N`. Tracks registry (`tracks`, `trackClips`) cần hỗ trợ dynamic keys.
- **D-03:** Overlay tracks xuất hiện phía trên Main Track (dưới SUB track). Thứ tự render: `[overlay-N, ..., overlay-1, sub, main, tts, bgm]`.

### Insert Between Clips
- **D-04:** **Ripple insert** — khi kéo clip vào giữa 2 clip trên Main Track, các clip phía sau tự dồn ra để nhường chỗ (giống CapCut).
- **D-05:** Insert indicator: hiển thị vạch xanh dọc tại vị trí sẽ chèn khi hover giữa 2 clip.

### Ghost Preview
- **D-06:** Ghost preview dạng **hình chữ nhật mờ** (semi-transparent rectangle) theo con chuột khi đang kéo. Không render thumbnail/waveform trong ghost.
- **D-07:** Track đích sáng lên (highlight background) khi hover, báo hiệu nơi clip sẽ drop.

### Cross-Track Drag
- **D-08:** Cho phép kéo clip từ Main Track lên → tạo overlay track và di chuyển clip sang đó.
- **D-09:** Cho phép kéo clip từ overlay track xuống Main Track → insert vào Main Track, xóa overlay track nếu trống.
- **D-10:** Kéo audio clips giữa các audio tracks (BGM, TTS) tự do.

### Agent's Discretion
- Chi tiết implementation: dynamic track creation/deletion logic
- Insert detection algorithm (khoảng cách giữa 2 clip để trigger insert indicator)
- Auto-delete overlay track khi không còn clip nào
- Z-ordering cho overlay video rendering trong preview

</decisions>

## Existing Code Insights

### Cần sửa đổi
- `TrackId`: Hiện là `'main' | 'sub' | 'tts' | 'bgm'` — cần hỗ trợ `overlay-N`
- `TRACK_ORDER`: Hiện là `['sub', 'main', 'tts', 'bgm']` — cần dynamic
- `trackClips`: Hiện là `Record<TrackId, string[]>` — cần dynamic keys
- `useTimelineDrag`: Cần thêm insert detection, ghost state, cross-track logic

### Reusable Assets
- `appendClipToTrack()`: Có thể mở rộng cho dynamic tracks
- `rippleDelete()`: Logic dồn clip — tái sử dụng cho ripple insert
- `ClipBlock` component: Đã generic cho tất cả clip types
- Snap logic trong `useTimelineDrag`: Mở rộng thêm insert snap points

</code_context>

## Specific Ideas

- Ghost preview: rect mờ cùng kích thước clip, opacity 0.4, border dashed

- Track highlight: background tint khi hover, opacity 0.1
- Auto-cleanup: xóa overlay track trống sau 500ms delay (tránh flicker)
