1. Thêm logic xác định target track dựa trên mouseY:
```typescript
// Tính track tại vị trí mouseY
const getTrackAtY = (mouseY: number): string | null => {
  const rulerHeight = 26;
  let accumulatedHeight = rulerHeight;
  
  for (const trackId of trackOrder) {
    const track = timelineStore.tracks[trackId];
    if (!track) continue;
    const trackBottom = accumulatedHeight + track.height;
    if (mouseY >= accumulatedHeight && mouseY < trackBottom) {
      return trackId;
    }
    accumulatedHeight = trackBottom;
  }
  
  // Nếu mouseY < ruler → phía trên tất cả tracks → signal tạo overlay mới
  if (mouseY < rulerHeight + (trackOrder.length > 4 ? 0 : 10)) {
    return '__new_overlay__';
  }
  return null;
};
```

2. Trong `handlePointerMove` khi `draggingClipId`:
- Gọi `getTrackAtY(mouseY)` để lấy `targetTrackId`
- Nếu `targetTrackId === '__new_overlay__'`:
  - Set `highlightTrackId = '__new_overlay__'`
  - Ghost hiện ở phía trên track đầu tiên
- Nếu `targetTrackId` khác track hiện tại → set `highlightTrackId = targetTrackId`

3. Trong `handlePointerUp`:
- Nếu `highlightTrackId === '__new_overlay__'`:
  - Gọi `timelineStore.createOverlayTrack()`
  - Gọi `timelineStore.moveClipToTrack(clipId, newOverlayId)`
- Nếu `highlightTrackId` khác source track:
  - Validate: cho phép video → overlay, overlay → main, audio → audio tracks
  - Gọi `moveClipToTrack(clipId, targetTrackId)` hoặc `rippleInsert` nếu có insertIndicator
- Schedule `cleanupEmptyOverlayTracks()` sau 500ms
</action>

<action>
Thêm validation function `canDropOnTrack` trong Timeline.tsx:

```typescript
function canDropOnTrack(clipType: ClipType, sourceTrackId: string, targetTrackId: string): boolean {
  if (sourceTrackId === targetTrackId) return false; // same track = reposition, không phải cross-track
  
  // Video clips: main ↔ overlay
  if (clipType === 'video') {
    return targetTrackId === 'main' || isOverlayTrack(targetTrackId) || targetTrackId === '__new_overlay__';
  }
  
  // Audio clips: bgm ↔ tts tự do
  if (clipType === 'audio') {
    return ['bgm', 'tts'].includes(targetTrackId);
  }
  
  // Subtitle: không cho phép cross-track drag
  if (clipType === 'subtitle') return false;
  
  return false;
}
```

Sử dụng trong `handlePointerMove`: chỉ set `highlightTrackId` khi `canDropOnTrack` return true.
Sử dụng trong `handlePointerUp`: chỉ thực hiện move khi `canDropOnTrack` return true.

Visual feedback khi KHÔNG cho phép drop:
- `highlightTrackId = null` (không highlight)
- Ghost opacity giảm xuống 0.2 và border color đỏ
</action>
