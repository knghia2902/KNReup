# Summary: Dynamic Overlay Track System

## What was built

Mở rộng hệ thống track cố định (main/sub/tts/bgm) sang hỗ trợ dynamic overlay tracks (`overlay-1`, `overlay-2`...). Code cũ dùng `TrackId` union type cố định và `TRACK_ORDER` mảng cố định. Giờ đã có:

1. **Dynamic TrackId type**: `FixedTrackId | OverlayTrackId` (template literal `overlay-${number}`)
2. **`getTrackOrder()`**: Tạo track order động, overlay tracks nằm giữa SUB và MAIN
3. **Store CRUD**: `createOverlayTrack`, `removeOverlayTrack`, `cleanupEmptyOverlayTracks`, `moveClipToTrack`, `rippleInsert`
4. **Ghost preview**: Hình chữ nhật mờ (opacity 0.45, border dashed) theo chuột khi kéo clip
5. **Insert indicator**: Vạch xanh (#22c55e) pulsing tại vị trí sẽ chèn
6. **Cross-track validation**: `canDropOnTrack()` — video→main/overlay, audio→bgm/tts, subtitle→no cross
7. **Overlay drop zone**: Hiện khi đang kéo video clip, text "Thả để tạo Overlay Track"
8. **Track highlight**: Background `rgba(99,102,241,0.1)` với transition trên track đích
9. **TrackHeader overlay**: Stack icon + purple tint cho overlay tracks
