# Plan 08.4-01: Dynamic Overlay Track System

<objective>
Mở rộng hệ thống track cố định (main/sub/tts/bgm) sang hỗ trợ dynamic overlay tracks (overlay-1, overlay-2...). Overlay tracks nằm trên Main Track và có thể tạo/xóa runtime.
Covers: D-01, D-02, D-03
</objective>

<must_haves>
- TrackId type hỗ trợ string pattern `overlay-N` ngoài 4 fixed tracks
- Track registry (tracks, trackClips) hỗ trợ dynamic keys
- Track order render: `[overlay-N, ..., overlay-1, sub, main, tts, bgm]`
- TrackHeader hiển thị overlay tracks với icon phù hợp
- Overlay tracks tự xóa khi không còn clip (auto-cleanup sau 500ms delay)
</must_haves>

2. Đổi `TRACK_ORDER` thành hàm `getTrackOrder(overlayIds: OverlayTrackId[])`:
```typescript
export const FIXED_TRACK_ORDER: FixedTrackId[] = ['sub', 'main', 'tts', 'bgm'];

export function getTrackOrder(overlayIds: OverlayTrackId[]): TrackId[] {
  // Sort overlay IDs theo số (overlay-1, overlay-2...) rồi reverse
  const sorted = [...overlayIds].sort((a, b) => {
    const numA = parseInt(a.replace('overlay-', ''));
    const numB = parseInt(b.replace('overlay-', ''));
    return numB - numA; // overlay-N ở trên, overlay-1 ở dưới (gần main nhất)
  });
  return [...sorted, ...FIXED_TRACK_ORDER];
}
```

3. Thêm helper function:
```typescript
export function isOverlayTrack(trackId: string): trackId is OverlayTrackId {
  return trackId.startsWith('overlay-');
}
```

4. Cập nhật `TimelineState`:
```typescript
export interface TimelineState {
  tracks: Record<string, TrackMeta>;           // dynamic keys
  clips: Record<string, Clip | SubtitleClip>;
  trackClips: Record<string, string[]>;         // dynamic keys
  selectedClipId: string | null;
  zoom: number;
  scrollX: number;
  snapEnabled: boolean;
  snapThreshold: number;
  overlayTrackIds: OverlayTrackId[];           // NEW: ordered list of active overlay tracks
}
```
</action>

10. Cập nhật `setClips`: khởi tạo `trackClips` cho cả fixed tracks và existing overlay tracks.

11. Cập nhật `removeClip`: sau khi xóa clip, nếu track nguồn là overlay và `trackClips[trackId].length === 0`, schedule cleanup.
