Tạo `python-sidecar/app/engines/downloader/database.py`:

```python
import aiosqlite
import os
from datetime import datetime

DB_PATH = os.path.join(os.path.dirname(__file__), '..', '..', '..', 'data', 'downloads.db')

async def get_db():
    os.makedirs(os.path.dirname(DB_PATH), exist_ok=True)
    db = await aiosqlite.connect(DB_PATH)
    db.row_factory = aiosqlite.Row
    await db.execute("PRAGMA journal_mode=WAL")
    return db

async def init_db():
    db = await get_db()
    await db.executescript('''
        CREATE TABLE IF NOT EXISTS downloads (
            id INTEGER PRIMARY KEY AUTOINCREMENT,
            url TEXT NOT NULL UNIQUE,
            platform TEXT NOT NULL,
            video_id TEXT,
            title TEXT,
            uploader TEXT,
            duration INTEGER,
            thumbnail_url TEXT,
            file_path TEXT,
            file_size INTEGER,
            resolution TEXT,
            format_id TEXT,
            status TEXT DEFAULT 'pending',
            progress REAL DEFAULT 0,
            speed TEXT,
            error_message TEXT,
            created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
            completed_at TIMESTAMP,
            metadata TEXT
        );
        CREATE INDEX IF NOT EXISTS idx_downloads_status ON downloads(status);
        CREATE INDEX IF NOT EXISTS idx_downloads_platform ON downloads(platform);
        CREATE UNIQUE INDEX IF NOT EXISTS idx_downloads_url ON downloads(url);
    ''')
    await db.commit()
    await db.close()
```

Cung cấp async functions: `add_download()`, `update_download()`, `get_download()`, `list_downloads()`, `delete_download()`, `check_url_exists()`.

- Class `DownloadManager`:
  - `__init__(max_concurrent=2)`: Init `asyncio.Semaphore(max_concurrent)`, init engines (YtdlpDownloader, DouyinDownloader)
  - `async analyze(url)`: Detect platform từ URL → route đến đúng engine → trả metadata + formats
  - `async start_download(url, format_id, output_dir)`: 
    1. Check dedup qua `check_url_exists()`

    3. Spawn `asyncio.create_task(_download_worker(download_id))`
    4. Return download_id
  - `async _download_worker(download_id)`:
    1. `async with self.semaphore:`
    2. Update status='downloading'
    3. Gọi engine.download() với progress_callback cập nhật SQLite progress
    4. On success: status='completed', set file_path, completed_at
    5. On error: status='error', set error_message
  - `async get_status(download_id)`: Query SQLite
  - `async get_history(limit, offset, platform)`: Query SQLite với filter
  - `async delete_download(download_id, delete_file=False)`: Remove from DB + optional file delete
  - `async cancel_download(download_id)`: Cancel running task
  - Singleton instance qua module-level variable

Tạo `python-sidecar/app/routes/downloader.py`:

```python
router = APIRouter(prefix="/api/download")
```

Endpoints:
- `POST /api/download/analyze` — Body: `{"url": "..."}` → Trả metadata + formats list
- `POST /api/download/start` — Body: `{"url": "...", "format_id": "...", "output_dir": "..."}` → Start download, return download_id
- `GET /api/download/status/{download_id}` — Trả current progress/status
- `GET /api/download/stream/{download_id}` — SSE stream progress updates (giống pattern pipeline.py analyze endpoint)
- `GET /api/download/history` — Query params: `limit=50`, `offset=0`, `platform=all` → History list
- `DELETE /api/download/{download_id}` — Delete download record + optional file
- `POST /api/download/cancel/{download_id}` — Cancel running download
- `POST /api/download/cookie/sync` — Body: `{"browser": "edge"}` → Sync Douyin cookie
- `GET /api/download/cookie/status` — Check cookie validity

Register router trong `app/main.py`:
```python
from app.routes.downloader import router as downloader_router
app.include_router(downloader_router)
```
